# tmatrix

Simulate the matrix effect in your terminal

## Install
You can get the binary from [releases](https://github.com/fortio/tmatrix/releases)

Or just run
```
CGO_ENABLED=0 go install fortio.org/tmatrix@latest  # to install (in ~/go/bin typically) or just
CGO_ENABLED=0 go run fortio.org/tmatrix@latest  # to run without install

```

or
```
brew install fortio/tap/tmatrix
```

or
```
docker run -ti fortio/tmatrix
```


## Usage

```
tmatrix help

flags:
```
